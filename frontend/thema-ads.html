Excel Upload - Multi-Theme Support

Upload an Excel file (.xlsx) with the following columns:

Required Columns:
  • customer_id - Google Ads customer ID (without dashes)
  • theme - Theme name (see supported themes below)
Optional Columns:
  • ad_group_id - Ad group ID (if omitted, will auto-discover ad groups)
  • campaign_id - Campaign ID
  • campaign_name - Campaign name
  • ad_group_name - Ad group name

Auto-discovery mode: When ad_group_id is not provided, the system will automatically discover all ad groups in HS/ campaigns for that customer (excluding ad groups with the theme's DONE label).

Upload Ad Groups - Flexible Themes

Upload a CSV file with customer_id (required) and optional ad_group_id / theme columns.
Two modes:
• Specific ad groups: customer_id + ad_group_id for targeted processing
• Auto-discover: customer_id only - finds all ad groups in HS/ campaigns without theme's DONE label

Theme options:
• With theme column: Different theme per row (e.g., customer_id,theme with values like 123456,black_friday)
• Without theme column: Uses dropdown selection below for all rows

Auto-cleanup: Respects 3 RSA limit per ad group. When at limit, removes:
  1) Paused theme ads first
  2) Active theme ads from other themes

Default theme (used when CSV doesn't have a 'theme' column)
Number of items per API batch query (1000-10000, default: 5000)
