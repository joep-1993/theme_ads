Auto-Discover Ad Groups

Automatically discover and process ad groups from your Google Ads account:
• Searches all configured customer accounts
• Finds campaigns starting with HS/
• Selects ad groups with ENABLED status
• Applies selected theme to all discovered ad groups

Select which theme to apply to discovered ad groups
Recommended: Start with 100-1000 for testing
Number of items per API batch query (1000-10000, default: 5000)
Max items per job - large discoveries split into multiple jobs (10k-100k, default: 50k)
Check-up & Verification

Verify previously processed ad groups and detect missing theme ads:
• Checks ad groups that were previously processed
• Verifies theme ads still exist in ad groups
• Detects ad groups where theme ads were removed
• Creates repair jobs for ad groups missing theme ads
• Helps maintain ad coverage across campaigns

Recommended: Start with 10-100 for testing
Number of items per API batch query (1000-10000, default: 5000)
Max items per repair job (10k-100k, default: 50k)
